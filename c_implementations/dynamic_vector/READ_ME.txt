Introduction
- - - - - - -

This folder contains 4 folders:
include	- API file
libs	- shared library file
obj	- an object file
dv	- source code and a test file



How to compile?
- - - - - - - - 

Enter "dynamic_vector" folder, open the terminal on the current folder and use the makefile commands:
"make test"	- creates files in debug mode and run the executable.
"make all"	- creates files in debug and realse mode and run the executable.
"make vlg"	- create a log file with the output of the vlg command.
"make clean"	- delete all files created.
