Introduction

This folder contains:
stack folder	- stack API and source code

makefile	- to build and run the executable 



How to compile?
- - - - - - - - 

open the terminal on the current folder and use the makefile commands:
"make all"	- creates the obj and executable files.
"make run"	- creates the obj files and run executable.
"make clean"	- delete all files created.
